# Claude Code Instructions

See [AGENTS.md](AGENTS.md) for library usage guidelines, public API reference, and development instructions.

## Build Commands
- Build: `xcodebuild build -scheme SwiftVerificarValidation -destination 'platform=macOS'`
- Test: `xcodebuild test -scheme SwiftVerificarValidation -destination 'platform=macOS'`
- NEVER use `swift build` or `swift test`
